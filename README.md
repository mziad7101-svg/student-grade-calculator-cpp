# Student Grade Calculator (C++)

A simple C++ project that calculates the average grade of subjects entered by the user.

## Features
- Enter multiple subject grades
- Calculate average grade
- Uses loops and variables

## Concepts Used
- C++
- Loops
- Variables
- Input / Output

## Example Output

Enter number of subjects: 3

Enter grade 1: 80
Enter grade 2: 90
Enter grade 3: 70

Average = 80
